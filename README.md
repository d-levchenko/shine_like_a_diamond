## 💎 Handmade Jewelry Website

A responsive website for a handmade jewelry brand. The project was made in
collaboration with other developers that showcases products, brand information,
and provides a simple and elegant interface for users to explore jewelry
collections.

## 🔗 Live Page

Check our live page here:
[View Project](https://d-levchenko.github.io/HandmadeJewelery/)

## ✨ Features

- Responsive layout for desktop, tablet, and mobile
- Modern and clean UI design
- Hero section with brand introduction
- Product gallery section
- Benefits section
- Contact form for customer inquiries
- Optimized images and SVG icons
- Semantic HTML structure

## 🛠 Technologies Used

- **HTML5 / CSS3** (Flexbox / Responsive Design)
- SVG Sprite Icons
- Responsive Images (1x / 2x)
- **Vite** - Local development server. It uses Rollupand esbuild internally for
  bundling.

## 🚀 You can download the project and check it

1. Clone the repository:
   ```bash
   git clone https://github.com/d-levchenko/HandmadeJewelery.git
   ```
2. Install Node.js for your system
   ```bash
    https://nodejs.org/en/download
   ```
3. Use npm commands to start the project locally
   1. Install the dependencies

   ```bash
       npm install
   ```

   2. Start the project locally

   ```bash
       npm run dev
   ```

   3. You should see something like - https://localhost:5173 in your console to
      navigate on project page
