💎 Handmade Jewelry Website

A responsive website for a handmade jewelry brand. The project showcases
products, brand information, and provides a simple and elegant interface for
users to explore jewelry collections.

✨ Features

Responsive layout for desktop, tablet, and mobile

Modern and clean UI design

Product gallery section

Hero section with brand introduction

Benefits section

Contact form for customer inquiries

Optimized images and SVG icons

Semantic HTML structure

🛠 Technologies Used

HTML5

CSS3 (Flexbox / Responsive Design)

SVG Sprite Icons

Responsive Images (1x / 2x)
